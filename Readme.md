# 🌤️ Pro Interactive Weather Dashboard

A premium, modern, and interactive weather web application designed to provide real-time weather updates, 7-day forecasts, and hourly predictions with sleek animations and a high-end UI/UX.

## 🚀 Key Features

*   **Live Location & Search:** Get weather data for your current location using the Geolocation API or search for any city globally.
*   **Contextual Animations:** Dynamic particle system (Sun, Rain, Snow, Clouds) that updates the background based on real-time weather conditions.
*   **Interactive Forecasts:** 
    *   **Hourly Forecast:** View the temperature trend for the next 24 hours.
    *   **7-Day Forecast:** Detailed overview for the upcoming week.
*   **Day-Specific Details:** Click on any day to see detailed insights including Max/Min Temperature, Wind Speed, and UV Index.
*   **Unit Toggle:** Switch seamlessly between Celsius (°C) and Fahrenheit (°F).
*   **Modern UI/UX:** Built with Glassmorphism design principles, smooth staggered animations, and a responsive layout.
*   **Premium Splash Screen:** A beautiful animated "Welcome" interface.

## 🛠️ Tech Stack

*   **HTML5:** Structured and semantic markup.
*   **CSS3:** Glassmorphism, Flexbox/Grid, and CSS Animations (Keyframes).
*   **JavaScript (ES6+):** Async/Await for API fetching, DOM manipulation, and efficient State Management.
*   **API:** [Open-Meteo API](https://open-meteo.com/) (Reliable, high-accuracy, no API key required).
